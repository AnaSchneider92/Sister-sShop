# 🛍️ Sisters Shop – Loja Virtual

Projeto de uma loja virtual desenvolvido com foco na apresentação de produtos, navegação intuitiva e experiência do usuário.

---

## 📌 Sobre o projeto

O **Sisters Shop** é um site de e-commerce desenvolvido para simular uma loja online, permitindo a visualização de produtos de forma organizada e atrativa.

O objetivo é oferecer uma interface simples, moderna e funcional, facilitando a navegação e destacando os itens disponíveis.

---

## ✨ Funcionalidades

* 🛒 Exibição de produtos
* 📂 Organização por categorias
* 🔎 Navegação simples e intuitiva
* 💻 Layout responsivo
* 🎯 Interface visual atrativa

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

